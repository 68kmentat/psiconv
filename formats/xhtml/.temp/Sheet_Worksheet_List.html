Sheet Worksheet List

It seems likely there can be more than one worksheet in the file format, though there is always only one found (so we have 02 followed by a single «Sheet Worksheet List Element» instead of a «XListE» in actual documents). All cell references at other places always have an unknown byte containing 00 associated with them: this is probably meant to specify the worksheet number.

Data Description

B Always 02?

«XListE» A list of «Sheet Worksheet List Elements»
