Clip Art File

This file type is used by Sketch for its internal Clip Art collection, but also to hold pictures used by an application. It contains several pictures.

The Clip Art File starts as below:

0000 ID Always 41 00 00 10

0004 LListL List of offsets of each Clip Art Item

Clip Art Item

Each item is laid out as follows:

0000 ID Always 40 00 00 10

0004 L Always 02 00 00 00 ?

0008 L Always 00 00 00 00 ?

000C L Always 00 00 00 00 ?

0010 L Unknown. 0C 00 00 00 for Sketch Clip Art, 08 00 00 00 at other places.

0014 Paint Data Section
